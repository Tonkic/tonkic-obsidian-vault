#### 当导入一个模块时，会执行这个模块
```python
main.py:
import module

module.py:
print("success1")

out:
success1
```
#### 导入一个包时，会执行包的__init__.py
```python
main.py:
import package

package/__init__.py：
from . import module
__ALL__=[module] # 仅影响 `from package import *` 的行为
print("success2")

package/module.py:
print("success1")

out:
success1
success2
```
