### 原理
##### 任何高阶常微分方程都可以写成一组一阶常微分方程
令x1=y，x2=y'，x3=y''等，这样，原来的高阶方程就变成了 n 个一阶方程：  
x₁' = x₂  
x₂' = x₃  
...  
xₙ' = f(xₙ, ..., x₁, t)
所以模型只需要学习一组一阶常微分方程
### symbolic regression (符号回归)
#### Functional SR
例：输入x=(1,2,3,4,5)，y=(2,4,6,8,10)，输出y=2x
从 (x,g​(x)) 中推断函数 g​(x)
#### dynamical SR
例：想象一个质点在二维平面
运动，只能采样运动轨迹$(x_1,x_2,t)$,
然后推断出质点运动的速度$(x ̇_(1, ) x ̇_2)$
其中 x​(t) 的观测可能存在噪声，而 t 也可能是不规则采样的。
